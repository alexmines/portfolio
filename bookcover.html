Book Cover

The goal of this project was to redesign a past project. By redesigning my past work I was able to think further about how I could make this attractive to the public.

Overall

The purpose of this project was to take a book cover that already existed at a book store and redesign it how we thought could attract more readers. My method to pursuing this was by using bright colours and large text.

Type

The font faced used for this project was Abadi MT Condensed in Extra Bold. In the project the type is one of the main focuses considering there are no graphics. The type sets the tone of the entire book.

Colour

Choosing a bold colour palette made the piece stand out more. Pairing a bright blue with a subtle pink, grey and white let the type speak for itself.
